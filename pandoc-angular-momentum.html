Angular momentum

The general theory of angular momentum

The starting point is
[Ji, Jj]=iℏϵijkJk

There are two valid views: (1) we simply take it as true, and move on to derive other relations. (2) The generator of rotation is an observable called angular momentum, but this relies on classical mechanics and at best an analog. Whichever way we choose, this is a good starting point.

To derive the eigenvalues of Ji, we define
J2 = Jx2 + Jy2 + Jz2


J± = Jx ± iJy

See that J+† = J−, and J−† = J+.

To make progress we need to guess a few relations: we must understand the meaning of J±|jm⟩.

Find the J−J+ and J−J+ in terms of J2 and Lz, which is quite easy if we use the basic definitions. This has to do with the norm of J±|jm⟩.

With the basic definitions, we derive
$$\begin{aligned} {{[{J_{z}} , {J_{\pm}} ]}} &=& \pm\hbar J_{\pm} \nonumber\\ {{[{J_{+}} , {J_{-}} ]}} &=& 2 \hbar J_{z} \nonumber \\ {{[{J_+} , {J_-} ]_+}} & = & J_+ J_- + J_- J_+ \nonumber\\ &=& 2(J^2 - J_z^2)\nonumber\end{aligned}$$

We have three operators Jx, Jy, and Jz. There are connected by the commutation relations. So can we see that there are essentially two ‘independent’ variables. These are J2 and Jz. To proceed with the algebra, we retain Jz and use J− and J+.

We want to show that the possible values of j are
$$j = \frac{n}{2}, \ \ \ n \ge 0$$

To achieve this, we need
[J2, J]=0
The vector J means we have three relations
[J2, Jx]=[J2, Jy]=[J2, Jz]=0

Assume the common eigenket of J2 and Jz is |β, m⟩ such that
$$\begin{aligned} J^2 | \beta,m \rangle &=& \hbar^2 \beta | \beta,m \rangle \nonumber \\ J_z | \beta,m \rangle &=& \hbar m | \beta,m \rangle \nonumber \\\end{aligned}$$

Use (see Sakurai’s QM, p. 14)
⟨ϕ|ϕ⟩≥0
deduce
β ≥ m2

Next consider
J+|β, m⟩
where
JzJ+|β, m⟩=ℏ(m + 1)J+|β, m⟩

For a fixed β, there is a maximum m, denoted by j such that
J+|β, j⟩=0

Next consider
J−J+ = J2 − Jz2 − ℏJz
and
J−J+|β, j⟩=ℏ2(β − j2 − j)|β, j⟩=0
to give
β = j(j + 1)

Similarly, we can show that
JzJ−|β, m⟩=ℏ(m − 1)J−|β, m⟩
showing that J−|β, m⟩ is either an eigenstate of Jz with eigenvalue ℏ(m − 1) or a zero.

According to Balentine, β = j(j + 1)=k(k − 1). We can solve k in terms of j, i.e., j2 − k2 + j + k = (j − k + 1))(j + k)=0, implying that either k = −j or k = j + 1. Since j ≥ k, we must have k = −j as the valid solutition. We also have β = j(j + 1)≥m2, so j ≤ 0 (rather than j ≤ −1). These are fine points that all textbooks do not discuss.

The lower limit of m is set to be equal to k. So we have
J−|β, k⟩=0

Since
−j ≤ m ≤ j
in steps of one, we must have
j − ( − j)+1 = n
where n is an integer ≥1. So the allowed values of j are (n − 1)/2, which is multiple of 1/2. When n = 1, j = 0, when n = 2, j = 1/2, etc.

A practical formula in AM:
$$J_{\pm} { {| jm \rangle }} = \hbar \sqrt{ j(j+1) - m(m\pm1) } { {| jm\pm 1 \rangle }}$$

Note that
$$\begin{aligned} j(j+1) - m(m\pm 1) &=& (j-m)(j+m) + (j \mp m) \\ &=& (j\mp m)(j\pm m + 1)\end{aligned}$$
